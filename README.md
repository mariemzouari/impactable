# ImpactAble

## Description

**ImpactAble** is a social impact web platform that aims to **connect citizens with positive and accessible social impact opportunities**. The project addresses the need to centralize, within a single platform, different forms of citizen engagement such as events, offers, donation campaigns, complaints, and community discussions.

### Project Objective

ImpactAble's mission is to:
- Foster citizen and solidarity engagement
- Provide a centralized space for social impact initiatives
- Facilitate user participation in concrete social actions

### Main Features

- **User Management**: Registration, login, and personalized profile management
- **Event Management**: Creation and participation in social impact events
- **Offers**: Distribution and consultation of engagement opportunities
- **Donation Campaigns**: Financial support for social causes
- **Complaints**: Reporting and tracking system for issues
- **Discussion Forum**: Space for exchange and sharing among citizens

## Installation

To install and configure the **ImpactAble** project locally, follow the steps below:

1. **Clone the repository**

   ```bash
   git clone https://github.com/mariemzouari/impactable
   cd impactable
   ```

2. **Install a local server**

   - Download and install [XAMPP](https://www.apachefriends.org/), [WAMP](https://www.wampserver.com/), or [Laragon](https://laragon.org/)
   - Activate the **Apache** and **MySQL** services

3. **Configure the database**

   - Open **phpMyAdmin** via `http://localhost/phpmyadmin`
   - Create a new database named `impactable`
   - Import the SQL file provided in the project

4. **Deploy the project**

   - Copy the project folder to the `htdocs` directory (XAMPP/WAMP) or `www` (Laragon)
   - Ensure read/write permissions are properly configured

5. **Launch the application**

   Access the project via your browser:
   ```
   http://localhost/impactable
   ```

## Usage

### Prerequisites

- **PHP** version 8.1 or higher ([Download PHP](https://www.php.net))
- **MySQL** version 5.7 or higher
- **Apache** (included in XAMPP/WAMP/Laragon)

### Quick Start

1. Create a user account via the registration page
2. Log in to the platform with your credentials
3. Explore the different available modules:
   - **Events**: View and participate in social impact events
   - **Offers**: Discover engagement opportunities
   - **Donation Campaigns**: Support causes that matter to you
   - **Complaints**: Report issues in your community
   - **Forum**: Exchange with other engaged citizens
